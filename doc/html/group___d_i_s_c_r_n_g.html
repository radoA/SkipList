Detailed Description

Discrete random numbers generator for skip-lists.

Generates discrete random numbers following the probability law described in rng_get_value()

See Also
rng_get_value()

unsigned int rng_get_value ( RNG *  rng,
unsigned int  max_value 
)

Generate a new value in the range [0..max_value]. Values are generated such that they follow the discrete probability law defined by :

\( (0 \le i < max\_value) \rightarrow P(i) = \frac{1}{2^{i+1}} \)
\( p(max\_value) = \frac{1}{2^{max\_value}} \)
Parameters
rng: the seed describing the sequence.
max_value: the upper bound of the generated values
Returns
the generated number.
